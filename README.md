# Retail Sales EDA

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset.

## Files Included
- Retail_Sales_EDA.ipynb — Notebook with data cleaning and visualizations
- retail_sales_dataset.csv — Dataset used for the analysis
- images/ — Folder containing charts (optional)

## What’s Done
- Data cleaning
- Data visualization
- Top categories analysis
- Daily and monthly trends
